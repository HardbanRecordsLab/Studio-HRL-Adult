## 🚀 Szybki Start

### Wymagania:
- Przeglądarka wspierająca HTML5, CSS3, JavaScript
- Rozdzielczość minimalna: 320px (mobile)
- Zalecana: 1920px+ (desktop)

### Instalacja:
1. Pobierz wszystkie pliki projektu
2. Umieść je na serwerze WWW
3. Upewnij się, że struktura folderów jest zachowana
4. Otwórz `index.html` w przeglądarce

### Konfiguracja:
- Edytuj dane kontaktowe w sekcji `<footer>`
- Dostosuj formularz castingowy do potrzeb
- Skonfiguruj linki do mediów (audio/video)
- Personalizuj kolory i czcionki w CSS variables
