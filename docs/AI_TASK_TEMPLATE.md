# AI Task Template

Use this template when assigning work to any AI assistant.

```text
Task:
[Describe exactly what should be done.]

Goal:
[Describe the user-visible or business-visible outcome.]

Context:
- Read: docs/AI_PROJECT_CONTEXT.md
- Read: docs/AI_WORKFLOW.md
- Read: docs/AI_RULES.md
- Relevant files: [list paths]

Constraints:
- [constraint 1]
- [constraint 2]
- [constraint 3]

Definition of done:
- [done item 1]
- [done item 2]
- [done item 3]

Verification:
- [what should be checked]

Output format:
- Summary
- Files changed
- Verification
- Risks / follow-up
```
